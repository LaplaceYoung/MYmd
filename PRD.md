# 【PRD】轻量级 Markdown 桌面端阅读与编辑器

## 1. 产品概述
本项目旨在开发一款面向 Windows 平台的轻量级桌面端 Markdown 应用程序。该应用以“阅读体验优先、兼顾高效编辑”为核心目标，解决传统重度 IDE 打开 Markdown 文件过慢、预览流程繁琐的痛点。产品需实现“双击即开、所见即所得”，并融合 Microsoft Word 的 UI 风格与 Typora 的无缝编辑体验。

## 2. 系统与架构要求
*   **目标操作系统：** Windows（优先适配 Windows 10/11）。
*   **系统级集成：** 需注册为 Windows 系统的 `.md` 文件默认打开程序。支持通过双击本地 `.md` 文件直接唤起应用并渲染文档内容。

## 3. 核心功能需求 (Functional Requirements)

### 3.1 视图与编辑模式切换
*   **默认模式（类 Typora 所见即所得体验）：**
    *   默认进入沉浸式渲染视图。
    *   **无缝源码交互：** 用户光标点击指定段落或元素时，该部分自动展开为 Markdown 源码以供修改；光标移开（失焦）后，立刻实时渲染为排版后的视觉效果。
*   **分屏模式（Split-screen Mode）：**
    *   支持通过快捷键或视图菜单切换至“左右分屏”模式（左侧为纯 Markdown 源码编辑器，右侧为实时渲染预览区）。

### 3.2 顶部格式工具栏与快捷输入
*   **常驻工具栏（类 Word 体验）：** 
    *   界面顶部需保留一个常驻的格式工具栏（UI 布局参考 Windows Word 的精简版 Ribbon 菜单）。
    *   包含常用功能按钮：标题层级切换、加粗、斜体、删除线、无序/有序列表、插入引用、插入代码块、插入图片、插入表格等。
*   **全键盘操作支持：**
    *   完全支持原生 Markdown 语法直接输入并自动转换为富文本格式。
    *   支持标准的富文本快捷键（如 `Ctrl+B` 加粗，`Ctrl+I` 斜体等）。

### 3.3 Markdown 扩展语法与高级渲染支持
除标准 Markdown 语法外，解析引擎必须支持以下高级扩展：
*   **代码块与高亮（Code Highlighting）：** 支持常见编程语言的语法高亮。
*   **数学公式：** 支持 LaTeX / KaTeX 语法的数学公式渲染。
*   **图表支持：** 集成 Mermaid.js，支持渲染流程图、时序图、甘特图等。
*   **任务列表（Task Lists）：** 支持带有复选框的待办事项列表，且允许在渲染状态下直接点击切换勾选状态，并反向同步修改源码。
*   **表格高级交互：** 提供快速插入表格的 UI 工具，并在渲染视图下支持表格的行列拖拽、增删等快捷编辑操作。

### 3.4 文件与多窗口管理
*   **多标签页视图（Multi-Tab Interface）：** 
    *   当应用处于开启状态时，双击外部多个 `.md` 文件，需在当前主窗口内以“新建标签页（Tab）”的形式打开，而非弹出多个独立窗口（类似主流浏览器或代码编辑器的标签页管理）。
