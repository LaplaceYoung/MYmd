PRODUCT CAPABILITIES

不是“写字板”，是你团队的文档交付中台。

多视图编辑能力

WYSIWYG / Source / Split 无缝切换，满足从快速记录到结构校验的完整路径。

  • Focus Mode 与 Typewriter Mode
  • 多标签与未保存状态保护
  • 右键与菜单栏操作一致性

本地文件工作流

支持打开、保存、另存为、导出 HTML。目录结构透明，可迁移、可备份、可长期维护。

技术文档增强

原生内置 Mermaid 与 KaTeX，搭配语法高亮与 TOC，让方案文档可读、可审、可复用。

工程化发布闭环

CI、Pages、Release 同步推进。每次迭代都能形成“功能更新 + 对外传播 + 安装分发”一体化链路。
