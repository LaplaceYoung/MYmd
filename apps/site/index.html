让 Markdown 成为可交付的生产工作流，而不只是一个文本文件。

MYmd 是面向高频写作、知识沉淀和产品文档协作的桌面编辑器。支持 WYSIWYG/源码分屏、多标签、文件树、Mermaid/KaTeX、自动保存与系统关联打开。

核心能力

编辑与视图

WYSIWYG / Source / Split 三视图切换，支持 Focus Mode 与 Typewriter Mode。

文件工作流

多标签、新建/打开/保存/另存为/导出 HTML，Workspace 文件树与自动保存。

内容增强

Mermaid、KaTeX、语法高亮、TOC 目录、全局 Search/Replace 全链路可用。

桌面集成

单实例转发、文件关联打开（.md/.markdown）、自定义标题栏与窗口控制。

为什么是 MYmd

轻量桌面性能

基于 Tauri v2 + Rust 后端，启动与资源占用更克制，适合全天候常驻写作。

本地优先与可控

内容默认在本地文件系统，目录结构可迁移、可备份、可被任意 Markdown 工具继续使用。

可演进架构

以 Tauri + React + Zustand 为基础，天然适合后续接入 AI 辅助编辑与知识检索能力。

上线优化路线图（2026 Q2）

  1. 工程治理 CI 门禁、Issue/PR 模板、版本化发布模板、回归脚本标准化。
  2. 效率功能 命令面板（Ctrl+K）、全局快速打开、模板中心、会话恢复。
  3. 生态能力 插件机制最小可用版、可选自动更新、公开 Roadmap 和社区投票机制。
